# CloudResources

使用 CloudKit 存储项目中不常用的资源文件以减小包体积，并且可以动态替换。

## CloudResources

集成在工程中用于加载资源文件

### CocoaPods

```bash
pod 'CloudResources', :git => 'https://github.com/cezres/CloudResources.git'
```
